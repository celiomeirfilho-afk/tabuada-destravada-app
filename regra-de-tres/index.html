Regra de Três Simples

A Regra de Três Simples é usada para resolver problemas com duas grandezas proporcionais. Vamos aprender passo a passo!

Tipos de Proporção

📈 Proporção Direta

Uma grandeza aumenta → a outra também aumenta.

Ex: 5 cadernos = R$25 → 3 cadernos = ?

📉 Proporção Inversa

Uma grandeza aumenta → a outra diminui.

Ex: 4 operários = 10 dias → 8 operários = ?

Como Montar

A / B = C / D

Organize os valores conhecidos e descubra o desconhecido (D).

5 cadernos custam R$25,00. Quanto custam 3 cadernos?

A = 5 | B = 25 | C = 3 | D = ?

Como Resolver

1 Identifique se é direta ou inversa
2 Monte a proporção: A/B = C/D
3 Direta: D = (B × C) / A
4 Inversa: D = (A × B) / C
