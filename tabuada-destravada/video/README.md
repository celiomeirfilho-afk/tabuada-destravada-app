# GUIA RÁPIDO DE PRODUÇÃO — VSL Tabuada Destravada™

## FLUXO RESUMIDO

```
1. ROTEIRO → Ler este arquivo + roteiro-vsl.md
   │
2. VEO 3 → Gerar 40 clips (prompts-veo3.md)
   │
3. ELEVENLABS → Gerar 9 áudios de narração (prompts-elevenlabs.md)
   │
4. SUNO/UDIO → Gerar 4 faixas音乐 (prompts-musica.md)
   │
5. MOCKUP → Gravar tela do mockup HTML (mockup-app.html)
   │
6. CAPCUT → Montar tudo (guia-capcut.md)
   │
7. YOUTUBE → Upload como "Não listado"
   │
8. KIWIFY → Colocar link no checkout
```

## TEMPO ESTIMADO POR ETAPA

| Etapa | Ferramenta | Tempo |
|-------|-----------|-------|
| Roteiro | Leitura | 15 min |
| 40 clips | Google Veo 3 | 2-4 horas (geração + seleção) |
| Narração | ElevenLabs | 30 min |
| Música | Suno/Udio | 30 min |
| Mockup | Gravação de tela | 20 min |
| Edição | CapCut | 3-5 horas |
| Upload | YouTube + Kiwify | 30 min |
| **TOTAL** | | **~8-12 horas** |

## ORDEM DE TRABALHO RECOMENDADA

### Dia 1: Gerar conteúdo (4-5h)
1. Gere os 40 clips no Veo 3 (2-3h)
2. Gere a narração no ElevenLabs (30min)
3. Gere a música no Suno (30min)
4. Grave o mockup HTML (20min)

### Dia 2: Montar vídeo (4-5h)
5. Monte tudo no CapCut (3-5h)
6. Exporte o MP4

### Dia 3: Publicar (30min)
7. Upload no YouTube (unlisted)
8. Configure no Kiwify

## DICA IMPORTANTE

O **mockup HTML** (`mockup-app.html`) é uma página interativa que mostra as principais telas do app. Abra no navegador, maximize, e grave a tela enquanto navega pelas telas. Isso dá footage real do app para intercalar com os clips cinematográficos do Veo 3.
